# Packet-Tracer_exercicio-VI

Objetivo:
- Criação de 3 VLAN.
Neste exercicio foram utilizados os seguintes dispositivos:
- 4 Switchs 2960
- 9 pcs (hosts) distribuídos em 3
andares de um prédio.
- Neste projeto, todos os
computadores estão em uma
mesma rede.
